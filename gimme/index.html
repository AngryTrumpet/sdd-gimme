AngulaJS Demos

Purpose of this Website App

This website app shows a series of demos on how to implement CRUD (Create, Read,Update and Delete to a LocalStorage datasource) using AngularJS.

Look and feel

This website app is constructed using JQuery and JQuery Mobile frameworks for the layout, navigation and style.
